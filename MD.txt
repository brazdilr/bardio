# BARDIO.CZ - Projektová dokumentace

## 📋 Přehled projektu

**Název:** Bardio.cz  
**Typ:** Webová stránka pro službu tvorby osobních písní na míru  
**Datum vytvoření:** 2024  
**Stav:** HTML/CSS/JS verze dokončena - připraveno k deployi  

---

## 🎯 Cíl projektu

Vytvořit profesionální webovou prezentaci služby Bardio, která nabízí tvorbu osobních písní na míru. Web má sloužit jako hlavní marketingový nástroj s důrazem na konverzi návštěvníků na zákazníky.

### Klíčové cíle:
- Prezentace služby tvorby písní na míru
- Maximální konverze návštěvníků na zákazníky
- Vysoká rychlost načítání a SEO optimalizace
- Snadná údržba a možnost budoucího rozšíření
- Profesionální a hravý vizuální styl

---

## 🏗️ Technická architektura - FINÁLNÍ VOLBA

### Volba technologie: **HTML/CSS/JavaScript (Statický web)**

**Důvody výběru:**
- ⚡ **Nejjednodušší deploy** - stačí nahrát 3 soubory
- 🔍 **Perfektní SEO** - čistý HTML, rychlé indexování
- 📱 **Responzivní design** - CSS Grid a Flexbox
- 🎵 **Audio player** - Vanilla JavaScript
- 💰 **Levný hosting** - funguje všude, i zdarma
- 🛠️ **Jednoduchá údržba** - otevřít, upravit, uložit
- 🚀 **Okamžité výsledky** - žádné build procesy

### Proč ne Next.js:
- ❌ **Překomplikované** pro statický web
- ❌ **Složitý deploy** - build procesy, servery
- ❌ **Dražší hosting** - potřebuje Node.js
- ❌ **Pomalá údržba** - vyžaduje znalosti React/TypeScript
- ❌ **Overkill** - pro jednoduchý web zbytečné

### Struktura souborů:
```
bardio.cz/
├── index.html      # Hlavní HTML soubor
├── styles.css      # Všechny styly
├── script.js       # JavaScript funkcionalita
└── audio/          # Audio ukázky (volitelné)
```

## 🚀 **Deploy návod - HTML/CSS/JS verze**

### **Nejjednodušší deploy (5 minut):**

#### **1. GitHub Pages (Zdarma):**
```bash
# 1. Vytvořte GitHub repo
# 2. Nahrajte soubory z /simple-version/
# 3. Settings → Pages → Deploy
# 4. Hotovo! Web je na username.github.io/repo-name
```

#### **2. Netlify (Zdarma):**
```bash
# 1. Drag & drop složku /simple-version/ na netlify.com
# 2. Získejte URL (např. amazing-name-123456.netlify.app)
# 3. Připojte doménu v Domain settings
# 4. Hotovo!
```

#### **3. Vercel (Zdarma):**
```bash
# 1. Drag & drop složku /simple-version/ na vercel.com
# 2. Získejte URL
# 3. Připojte doménu
# 4. Hotovo!
```

### **DNS nastavení pro doménu:**
```
Type: CNAME
Name: www
Value: [your-hosting-url]

Type: A  
Name: @
Value: [hosting-ip]
```
